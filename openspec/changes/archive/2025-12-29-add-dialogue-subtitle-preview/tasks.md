## 1. 实施

- [x] 1.1 创建台词解析函数
  - 解析 `dialogue_prompt` 字段，提取时间范围和台词内容
  - 支持格式：`3～4秒:【角色描述】:"台词内容"` 或 `4～5秒:【角色描述】:"台词内容"`
  - 返回结构化的台词数据数组，包含开始时间、结束时间、角色描述和台词内容

- [x] 1.2 在预览面板HTML中添加字幕显示区域
  - 在 `preview-panel-content` 中添加字幕容器元素
  - 字幕容器位于视频/图片下方，居中显示

- [x] 1.3 添加字幕样式
  - 在 `episode-edit.css` 中添加字幕容器和字幕文本的样式
  - 字幕文本居中，字体大小适中，颜色清晰可见
  - 字幕容器有适当的间距和背景（可选）

- [x] 1.4 实现字幕显示逻辑
  - 在 `displayShotContent` 函数中，获取当前分镜的 `dialogue_prompt`
  - 如果存在台词提示词，解析并存储台词数据
  - 对于视频内容，监听视频的 `timeupdate` 事件，根据当前播放时间显示对应的字幕
  - 对于图片内容，根据图片显示的开始时间和持续时间，按时间顺序显示字幕

- [x] 1.5 实现字幕切换逻辑
  - 当视频播放时间进入某个台词的时间范围时，显示该台词
  - 当视频播放时间离开某个台词的时间范围时，隐藏该台词
  - 支持多个台词片段连续或重叠显示

- [x] 1.6 处理边界情况
  - 如果分镜没有 `dialogue_prompt`，不显示字幕区域
  - 如果台词解析失败，不显示字幕或显示错误提示
  - 如果视频播放被暂停或跳转，字幕同步更新
