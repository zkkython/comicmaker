# 剧集管理

## 修改需求

### 需求：作品详情编辑
系统必须允许用户编辑作品详情，包括：
- 作品名称
- 作品描述
- 封面图片（支持多张不同宽高比的图片）
- 风格描述（支持从已有风格选择）
- 默认视频宽高比（9:16、16:9、4:3、3:4）
- 素材关联（人物、场景、道具素材）

#### 场景：选择已有风格
- **当** 用户在作品详情编辑页面
- **且** 在风格描述字段旁边看到风格选择下拉框
- **且** 下拉框显示"选择风格..."选项和所有已有风格（按名称排序）
- **当** 用户从下拉框选择一个风格
- **则** 该风格对应的描述自动填充到风格描述字段
- **且** 用户可以基于填充的描述进行修改
- **且** 保存更改后，风格描述保存到 `meta.json`
- **且** 风格描述用作内容生成的上下文

#### 场景：编辑风格描述
- **当** 用户在作品详情编辑页面
- **且** 修改风格描述文本（无论是手动输入还是从风格选择填充）
- **且** 保存更改
- **则** 风格描述保存到 `meta.json`
- **且** 风格描述用作内容生成的上下文
